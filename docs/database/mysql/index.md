# My Sql
## 启停MySQL服务
注意：
net start mysql57和net stop mysql57命令中的 mysql57 是 MySQL 服务器名称，如果你的 MySQL 服务名称是 DB 或其它的名字，应该输入net start DB或其它名称，否则提示服务名无效。
### 启动
```bash
net start mysql57
```
### 停止
```bash
和net stop mysql57
```
